# Verbus
Small application for learning irregular verbs
Based on VIPER architecture

## Instruments stack:
* UIKit & Foundation:
    - Calendar;
    - UserDefaults;
    - UINotificationFeedbackGenerator;
    - UITableView & UICollectionView;
    - Custom views.
* CoreData:
    - NSFetchedResultsController;
    - NSPredicate & NSSortDescriptor;
    - NSBatchUpdate.
* AVFoundation:
    - AVSpeechSynthesizer.
* UserNotifications.
